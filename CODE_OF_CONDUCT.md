# Code of Conduct

## Our Pledge

We are committed to making participation in this project a welcoming, respectful, and collaborative experience for everyone.

Whether you are a beginner or an experienced developer, you are welcome to contribute.

---

## Expected Behavior

Contributors are expected to:

- Be respectful.
- Be constructive during discussions.
- Help beginners whenever possible.
- Respect different opinions and ideas.
- Give proper credit to authors and contributors.
- Follow the project license.

---

## Unacceptable Behavior

The following are not acceptable:

- Harassment or abusive language.
- Personal attacks.
- Discrimination.
- Plagiarism.
- Removing author or contributor credits.
- Commercial misuse of the project.
- Violating the project license.

---

## Community Philosophy

This project exists to:

- Help students learn Embedded Systems.
- Encourage Bare-Metal Programming.
- Build high-quality reusable drivers.
- Remain permanently free for everyone.

Knowledge grows when it is shared.

---

## Enforcement

Project maintainers may edit, reject, or remove contributions that violate this Code of Conduct or the project license.
